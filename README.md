# Xberry-Pi-2 : ZX Spectrum Next KS2 Clone
This is a ZX Spectrum Next KS2 Clone in the form of Raspberry Pi. It is 100% compatible with ZX Specrum Next KS2 (https://www.specnext.com/). PCB is the same size as Raspberry Pi Board, can be fit in Raspberry Pi 3+ or 4B case with small modification.

    PCB version 1,
    FPGA: XC7A15T-1CSG324C
    2MB RAM, 256MBIT Flash ROM
    RTC onboard with battery backup
    Header for ESP-8266-01 or ESP-8266-12F Module
    Micro HDMI and VGA port
    Audio and EAR/MIC connectors
    2 Joystick port
    PS/2 port for Keyboard/Mouse
    Header for connect to Matrix Keyboard
    Header for External SD and 2nd SD card
